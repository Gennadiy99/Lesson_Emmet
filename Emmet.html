    Сокращенное написание "ct" Создаст content Свойство CSS content применяется к псевдоэлементам (::before и ::after) для определения содержимого, которое будет вставлено перед или после содержимого основного элемента.

    Сокращенное написание "q" Создаст quotes - используется для определения кавычек, которые будут использоваться в свойстве content псевдоэлементов ::before и ::after.

    Сокращенное написание "ol" Создаст outline применяется к элементам, чтобы добавить контур вокруг них, который отображается вне границ элемента. В случае таблицы, это свойство может применяться к таблице в целом или к отдельным ячейкам, строкам или столбцам

    Сокращенное написание "tb:f" Создаст table-layout: fixed; - означает, что ширина колонок таблицы определяется шириной ячеек заголовка (thead) или первой строки данных (tbody). В этом режиме ширина колонок остается постоянной независимо от содержимого ячеек внутри них или изменений в размерах окна браузера.

    Для работы с списками

    Сокращенное значение "lis" - Создаст list-style: ; Свойство CSS list-style используется для управления стилями маркеров (bullets или числовых значений) в списке.

    Сокращенное значение "lis:n" - Создаст list-style: none; Свойство list-style: none; в CSS используется для скрытия маркеров (bullets или числовых значений) в списке. Это полезно, когда вы хотите создать список без маркеров.

    Сокращенное значение "lisp:i" - Создаст list-style-position: inside; Когда вы устанавливаете list-style-position: inside;, маркеры списка отображаются внутри контейнера элемента списка, рядом с текстом. Это означает, что текст начинается сразу после маркера, а не под ним.

    Сокращенное значение "list" - Создаст list-style-type: ; Если вы оставляете list-style-type:; пустым, это означает, что вы не задаете конкретный стиль маркера, и браузер будет использовать свои стандартные стили маркеров в зависимости от типа списка (ненумерованный или нумерованный) и стиля браузера.

    Сокращенное значение "list:c" - Создаст list-style-type: circle; Свойство list-style-type: circle; в CSS устанавливает стиль маркера списка в виде круга c пустотой в нутри.

    Сокращенное значение "lisi" - Создаст list-style-image: ; Далее нужно будет указать url('bullet.png').В CSS используется для задания изображения в качестве маркера списка вместо стандартного браузерного маркера

    Сокращенное значение "@f" - Создаст @font-face { font-family: ; src: url(); } - Когда вы используете @font-face, вы определяете пользовательский шрифт, указывая его источник (обычно ссылку на файл шрифта на сервере) и дополнительные характеристики шрифта, такие как имя, начертание и т. д. После этого вы можете использовать этот шрифт в своем CSS, как любой другой шрифт.

    Сокращенное значение "@ff" - Создаст: @font-face {
    font-family: 'FontName';
    src: url('FileName.eot');
    src: url('FileName.eot?#iefix') format('embedded-opentype'),
    url('FileName.woff') format('woff'),

    url('FileName.svg#FontName') format('svg');
    font-style: normal;
    font-weight: normal;
    }
     - Создаст более развернутую структуру подключения шритов.

    Сокращенное значение "@i" - Создаст @import url(); - это правило в CSS, которое позволяет включать стили из одного CSS-файла в другой. С его помощью вы можете импортировать стили из внешнего источника, такого как другой CSS-файл, в текущий CSS-файл.
    Синтаксис написания: @import url('stylesheet.css');

    Сокращенное значение "@kf" - Создаст @keyframes identifier {} - Это правило в CSS, которое определяет анимацию, позволяя вам определить стили, которые должны изменяться во времени. Идентификатор (identifier) указывает на имя анимации, которое вы создаете.
    Пример использования:
    @keyframes slidein {
    from {
    transform: translateX(-100%);
    }
    to {
    transform: translateX(0);
    }
    }

    Сокращенное значение "@m" - Создаст @media screen- Это медиа-запрос в CSS, который применяется к стилям только при условии, что текущее устройство является экраном (например, монитор или мобильное устройство)
    Пример: @media screen and (max-width: 600px) {
    body {
    font-size: 14px;
    }
    }     В этом примере стили будут применяться только для устройств с экраном шириной не более 600px. Это позволяет создавать адаптивные веб-страницы, которые могут отображаться оптимально на различных устройствах и разрешениях экрана.

    Сокращенное значение "ac" - Создаст align-content: start;Свойство align-content в CSS используется для управления выравниванием контейнера вдоль оси блочной упаковки, когда в контейнере есть несколько рядов элементов. Значение start для свойства align-content указывает, что элементы в контейнере выравниваются в начале (верхнем крае) контейнера вдоль оси блочной упаковки.

    Сокращенное значение "fx" - Создаст flex: ;Свойство где можно выбрать одно из свойств flex-direction: ;

    Возможности Emmet c Методологией БЭМ

    Принцип Emmet с методологией БЭМ таков:

    Здесь в конце выражения используется фильтр - "|bem"

    • Указывается названия Тега - "form".
    • После Класс Блока - .search-form.
    • дальше если МОДИФИКАТОР то указывается Префикс+Класс Модиф "._" Класс Модификатора".
    • если Элемент то указывается Префикс ".-" Класс Элемента".
    • Везде где ставится "."подставляется Класс Блока автоматически.
    • Если после "." поставить "__Класс Элем". и сразу "_Класс Мод". То создастся Класс Элемента и Класс Модификатора Оба Кл.начинаются с назв Кл.Блока
    • Для этого в конце строки нужно установить Фильтр "|bem"

    Пример кода в Emmet:

    form.search-form._wide.>input.-qutry-string+input:s.-btn_large|bem

    Результат выполненного кода: (элементы и стили можно посмотреть в html документе)

    Стоит отметить что в названии Класса Элем. после "."
    ставится одно тире "-"
    если установить два "--"
    то название Класса Блока берется на уровень выше
    Если три тире "---" то название Класса Блока берется на 2 уровня выше.

    .b1>.b2_m1>.-e1+.---e2_m2|bem
    .lot_size_10._color_red|bem
    .lot>.-window_hiden|bem
    .header_font>.logo_size+.menu_size>.-item*5|bem
    .header_font>.logo_size._font+.menu_size>.-item*5|bem
